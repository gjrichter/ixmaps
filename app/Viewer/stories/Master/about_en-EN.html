Some information how to use the map application

interact with the map

There are two states for interaction with the map:

In the 'zoom/pan' mode you can use the mouse and the mouse wheel to zoom and pan the map, like in most other map applications.

In the 'info' mode you can evoke the display of map item information.

You can enter the 'info' mode by clicking either on a map object or on the specific icon in the upper left corner.

Clicking on the map background brings you back to the zoom/pan mode.


adapt the map theme

All map themes are defined by a project object and created in the browser. You can change the theme parameters by tools or using an online source editor to customize the theme.

The icon opens a tool bar to change the theme. The bar has several content layer. By clicking on the first element of the bar you can walk through the tools.

Furthermore there is a context menu (right mouse click) to open theme configuration tools, to share or save the customized theme.

You will find a description of this menu below.

save and share

Tools to save or share the actual map you find either in the top menu of map viewer or in a context menu.

With 'save project' of the context menu you can save the actual map as iXMaps Project (JSON) into a local file.

context menu

You can open the context menu clicking with the right mouse key on any point of the map. The menu offers several functions to analyze, customize, share or save the actual map.

open tools

opens a toolbar with instruments to customize the map theme

  edit theme

opens an online editor to inspect and change the JSON code that defines the actual map theme

  config theme

opens the iXMaps theme configurator loaded with the actual map theme 

  filter theme

opens a dialog to analyze the theme data and create facet filter based on the data columns.

  share map

opens a share dialog to obtain a sharable link to the actual map and theme or HTML embed code.

  open in editor

opens the iXMaps editor application in a new browser tab preloaded with the actual map theme.

  open in viewer

opens a copy of the actual map in the viewer application within a new browser tab

  copy theme / paste theme

because iXMaps themes are defined each by one JSON object, you can copied and paste a theme, may be to add it to another map

You can save the result, the map with two themes, into a new project.


  save project

You can save the actual map and theme into a local JSON file.

An iXMaps project defines all elements of an iXMaps thematic map in one JSON object.

With a project defined in JSON you can reproduce the thematic map on any browser loading the project into the iXMaps project viewer or an embed frame.
