Vectorgraphic mapping and charting framework

Publish multilayer geographic data with static SVG tiles and geopositioned vector graphic charts. All handled by a reactive, mobile compatible GUI in HTML5


'Life' theme editing

Every thematic map offers the user numerous editing facilities to adapt the theme representation to his needs. This includes also a theme code editor, realized with the ACE online code editor, to see, copy or manipulate directly the JavaScript object of the actual map theme.

Share or bookmark a customized theme.

You can share or bookmark your customized theme by an URL which iXMaps will create for you on demand. The sharing function gives you also a code snippet to embed the customized theme in an arbitrary HTML page.

Design incredible maps with Mapbox Studio

Anyone can design a map with Mapbox Studio. Our open source design tool allows you to choose what you want on the map and exactly how it looks. Add data, pick fonts and colors – you have control over every detail. Plans start at just $5/month.
